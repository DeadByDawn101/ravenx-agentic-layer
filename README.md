# RavenX Agentic Layer

A minimal scaffold for a practical agent orchestration layer in the RavenX workspace.

## Goals
- define a clean boundary between prompts, tools, skills, and execution loops
- keep the first iteration docs-first and easy to evolve
- provide a small entrypoint for future implementation

## Initial layout
- `ARCHITECTURE.md` outlines the intended structure and responsibilities
- `src/index.md` is a lightweight placeholder for the first runtime surface
- `Makefile` provides a couple of simple project commands

## Next steps
- turn the architecture notes into a thin executable prototype
- add a concrete runtime and interface contract
- add tests once the first code surface exists
